Contact Book Help Page

This application facilitates you to manage your contacts. Some of the features are:

Add/Update a contact
Users can add a new contact
Name is required field
Phone number should be 12 digits including country code
Save/Update button will remain disabled in case of invalid input values
Click on the ID of any pre existing contact from the list and update the values when they show up on the form.
Delete a contact
A cross sign appears at the end of each contact which can be used to delete a contact
Search/Sort a Contact
A search box facilitates searching any value from the complete contact book.
CLick on any column heading to sort ascending or descending . Click again to flip the sort direction.
